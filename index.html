About Me

Hello, I'm Anthony (Tony), a software developer and test technician. Here you can find a list of some of my most recent project contributions and game jam entries.

All of the projects described here are created in my own spare time.

I am also the creator of the CodeNMore YouTube channel, where I teach basic game programming and basic electronics skills. Currently the CodeNMore channel video production is on hold.

Open Source Projects / GitHub

Creator of:

CScreen

I created CScreen to provide an easy-to-use way for Love2D games to be written in a resolution-independent manner, providing aspect ratio management and letterboxing.

Sinote

A minimalistic note taking application written in Flutter. Supports creating notes manually or via instant-pasting from the clipboard, note categorization, reordering, and more.
